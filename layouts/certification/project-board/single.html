Certification SIG Project Board

How it Works

To foster collaborative decision-making and enhance transparency, we have established a dedicated GitHub board. This board provides an asynchronous method of communication, allowing for more flexible timelines, especially for international participants. Additionally, the GitHub platform serves as a mechanism for organizations to collaborate with the SIG on making certification requests from ISVs and IHVs.

Details: Templates

Our GitHub board features issue templates tailored for various Certification Activities, including:

  • running the Hardware Certification Suite
  • sending hardware to ALOSF labs.

In-Process Discussions

  • The GitHub Project Board facilitates issue-based dialogue between involved parties. Access is secured and requires authorization from the SIG lead.
  • GitHub Discussions for the Certification SIG Project serve as an open forum and a platform for FAQs (requires only a GitHub account).

How to Join and Act

  • Create a GitHub account.
  • Request access to the project board:
  • ask on GitHub Discussions
    •     OR
  • use the Contact form available here
    •     OR
  • via our Mattermost channel
  • Join the board and create new issue selecting from available templates.
