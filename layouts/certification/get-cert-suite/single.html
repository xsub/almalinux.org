About the Certification Suite

The AlmaLinux Certification Suite, formerly known as the Hardware Certification Suite (HCS), is a comprehensive set of tools and processes designed to validate the compatibility and performance of hardware with AlmaLinux OS. It includes a range of automated tests that cover various aspects of system functionality, from CPU and memory to peripherals and network performance.

Recently, the suite has been expanded to encompass ISV-specific testing extensions, allowing software vendors to certify their applications and services on AlmaLinux, ensuring seamless integration and operation. This expanded suite supports the goal of providing a robust certification process that verifies both hardware and software for stability, compatibility, and performance, reinforcing the platform's reliability for enterprise environments.

Hardware Detection

This test detects the hardware and collects basic information.

CPU Stress Testing

This test assesses the CPU's performance under stress conditions.

Containerization

This test evaluates the functionality and performance of containerization technologies.

KVM Functionality

This test examines the functionality of KVM (Kernel-based Virtual Machine).

Network Performance

This test assesses network performance and throughput.

Linux Kernel Testing through LTP (Linux Test Project)

This test involves kernel testing using the Linux Test Project (LTP).

USB Port Functionality

This test checks the functionality of USB ports on the system.

PXE Device Booting

This test verifies the functionality of PXE (Preboot Execution Environment) device booting.

OS Feature Benchmarking via PTS (Phoronix Test Suite)

This test benchmarks various OS features using the Phoronix Test Suite (PTS).
